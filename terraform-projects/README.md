# Terraform Projects

This folder contains Infrastructure as Code (IaC) projects built using Terraform.  
Each project demonstrates how to automate AWS resource provisioning such as VPCs, EC2 instances, and S3 buckets.

## Projects
- **VPC Setup Project** – Custom VPC with public/private subnets, IGW, and route tables.
- **EC2 Web Server Deployment** – Launch and configure EC2 instances with security groups.
- **S3 + IAM Automation** – Create and secure S3 buckets using Terraform modules.
